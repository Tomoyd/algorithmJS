#### 栈

栈是一种操作受限的线性数据结构

数组是一种数据结构，可以在任意位置插入删除元素

###### 受限操作：

- 后进先出,先进后出
- 仅允许在栈顶操作
- 出栈（退栈）栈顶元素删掉，并使其相邻的元素形成栈顶
- 入栈（压栈）称为新的栈顶

#### javascript 栈操作

javascript 对数组进行：

- push：入栈
- pop: 出栈并返回出栈元素

或者

- unshift 入栈
- shift 出栈

此外访问栈顶元素：`arr[arr.length-1]`

#### 队列

- 先进先出
- 队头出，队尾进
- 出队的一端队头
- 进队的一端队尾

#### javascript 操作配合

- push 入队
- shift 出队

或者

- unshift 入队
- pop 出队
